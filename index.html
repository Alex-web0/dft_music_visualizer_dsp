Discrete Fourier Transform (forward):

$$X[k] \;=\; \sum_{n=0}^{N-1} x[n]\, e^{-j\,2\pi k n / N}, \qquad k = 0, 1, \dots, N-1$$

Inverse Discrete Fourier Transform:

$$x[n] \;=\; \frac{1}{N}\sum_{k=0}^{N-1} X[k]\, e^{\,j\,2\pi k n / N}$$

Euler's identity (the bridge between exponentials and sinusoids):

$$e^{j\theta} \;=\; \cos\theta + j\sin\theta$$

Key properties used in this app:

  • Linearity: $a\,x_1[n] + b\,x_2[n] \;\longleftrightarrow\; a\,X_1[k] + b\,X_2[k]$
  • Time shift: $x[n - n_0] \;\longleftrightarrow\; X[k]\, e^{-j\,2\pi k n_0 / N}$
  • Frequency shift (modulation): $x[n]\, e^{\,j\,2\pi k_0 n / N} \;\longleftrightarrow\; X[k - k_0]$
  • Parseval (energy conservation): $\displaystyle\sum_{n=0}^{N-1} |x[n]|^2 \;=\; \frac{1}{N}\sum_{k=0}^{N-1} |X[k]|^2$
  • Hermitian symmetry (real input): $X[N-k] = \overline{X[k]}$
  • Convolution theorem: $x[n] * h[n] \;\longleftrightarrow\; X[k]\, H[k]$
